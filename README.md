## 起因
我在 Edge 浏览器浏览网站时，发现某外挂网站对我发起了大量攻击。一开始我自己也一头雾水，页面还会自动跳转到百度页面，当时实在十分无奈，于是我开启了侧边搜索。
之后我对该网站做了逆向还原，发现其中存在大量防御机制以及海量 AI 程序代码。我实在无法理解，外挂团队不专心开发外挂、维护自身网站
为何还要主动攻击用户、投放大量恶意垃圾代码。为了予以惩戒，我决定将相关内容开源，毕竟我也不清楚他们的外挂为何无法正常使用，不然原本还打算直接进行破解。
它到底做了什么简单说，这个网站在前端堆了一大堆 “防调试” 的东西：
禁右键、禁F12和各种快捷键、检测开发者工具有没有打开、控制台刷垃圾信息、一旦检测到你在调试，就直接跳转走、一些比较典型的没啥用的泔水，因为对于我来说真的不如好好做全栈好，AI 写的其实没必要
这种东西，说实话，只能防完全不懂的人。稍微会点的，根本拦不住。还有一个比较烦的点是控制台污染，比如不停刷一些无意义的内容，甚至直接改写 console.log。
这个确实会影响调试体验，但本质上也只是 “恶心你一下”，并不构成什么真正的防护。
## 最后是最直接的
一旦检测到你可能在调试，直接 window.location.href 给你跳走，拆完之后的感受说实话，整套东西看下来，有一个很明显的感觉就是真的很拉跨，我观察了两圈
想吐槽GTA的圈子（针对一些比较魔怔的，而不是说全部）外挂团队怎么部分那么专注网站啊？不做好，专心做网站不如转行得了，前端搞得不能看似的
一堆防F12，都用AI生成了，还用所谓的F12,没必要，好好做挂得了。。。而且我在b站看了下风评也不是很好，搞什么网页啊，你又不是做艺术的😅
